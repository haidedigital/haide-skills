# Commercial Judgment Skill

## Philosophy

**Revenue is vanity, profit is sanity, cash is king.** Price based on value created, not hours worked. Understand that saying "no" to wrong opportunities enables saying "yes" to right ones. Every deal must work for both sides.

## Core Financial Framework

### The Agency Economics Equation

```
Revenue - Cost of Delivery = Gross Profit
Gross Profit - Operating Expenses = Net Profit
Net Profit + Depreciation = EBITDA
```

**Target Metrics**:
- **Gross Margin**: 70%+ (delivery costs <30% of revenue)
- **Net Margin**: 20-30% (after all expenses)
- **Cash Buffer**: 3-6 months operating expenses
- **LTV:CAC**: 3:1 minimum (lifetime value : acquisition cost)
- **Payback Period**: <12 months (time to recover CAC)

## Value-Based Pricing

### The Value Calculation

**Never price based on**:
- Time/hours worked
- What competitors charge
- What you "think they can afford"
- Cost-plus margin

**Always price based on**:
- Economic value created for client
- Value of alternative solutions
- Client's willingness to pay
- Strategic value of relationship

---

### Value Pricing Formula

**Step 1: Calculate Client's Value at Stake**

```
Current State â†’ Desired State = Value Gap

Example (SEO for E-commerce):
Current: $500k/year organic revenue
Target: $2M/year organic revenue
Value Gap: $1.5M additional revenue

Your fee: 10-20% of value created

```

### The 10x Rule

**Your price should be 1/10th of the value you create.**

If you can't demonstrate 10x ROI, you're either:
1. Not creating enough value (wrong solution)
2. Not communicating value effectively (marketing problem)
3. Targeting wrong clients (qualification problem)

## Unit Economics

### Cost of Delivery

**Calculate true cost per client**:

```
Fixed Costs (per month):
+ Team salaries (account manager, specialists)   $15k
+ Tools/software (subscriptions)                 $2k
+ Overhead allocation (office, admin)            $3k
= Total Fixed Costs                              $20k

Variable Costs (per client/month):
+ Contractor/freelancer costs                    $1-3k
+ Platform fees (ads, tools)                     $0.5-2k
= Variable Cost per Client                       $1.5-5k

Total Cost for 5 Clients:
$20k fixed + ($3k variable Ã— 5) = $35k/month
```

---

### Margin Calculation

```
Revenue per Client:                              $10k
- Variable Cost:                                 $3k
= Contribution Margin:                           $7k (70%)

Total Revenue (5 clients):                       $50k
- Total Variable Costs:                          $15k
= Total Contribution Margin:                     $35k
- Fixed Costs:                                   $20k
= Net Profit:                                    $15k (30% net margin)
```

**Break-even**: Need 3 clients at $10k to cover fixed costs ($20k).

---

### Minimum Viable Price

**Formula**:
```
Minimum Price = (Fixed Costs + Desired Profit) / # of Clients / Gross Margin

Example:
($20k fixed + $10k profit) / 5 clients / 70% margin
= $30k / 5 / 0.7
= $8,571/month minimum
```

**Never go below your minimum** unless strategic loss leader.

## Cash Flow Management

### The Cash Flow Problem

**Agency cash flow gap**:
```
Day 1: Sign new client
Day 1-30: You pay team, tools, overhead ($10k)
Day 30: Send invoice to client ($12k)
Day 45: Client pays invoice (Net-15)

Cash gap: 45 days Ã— $10k costs = Need $15k buffer per client
```

---

### Cash Flow Solutions

**1. Upfront Payments**
```
Traditional: $10k/month, paid monthly = $10k upfront
Better: $30k quarterly, paid upfront = $30k upfront (3x buffer)
Best: $108k annually, paid upfront = $108k upfront (no gap)
```

**2. Deposits on Projects**
```
Project fee: $50k
50% deposit: $25k (covers costs immediately)
50% on completion: $25k (pure profit)
```

**3. Retainer + Performance**
```
$5k/month base (covers costs) + $5k performance bonus
Base paid 1st of month (guarantees cash flow)
Bonus paid after delivery (upside)
```

**4. Payment Terms**
```
Avoid: Net-30 or Net-60 (kills cash flow)
Use: Due on receipt or Net-7
Best: Auto-pay credit card on file (immediate)
```

---

### Cash Flow Formula

**Minimum cash buffer**:
```
Monthly Operating Expenses Ã— 3 months = Minimum Buffer

Example:
$40k/month Ã— 3 = $120k minimum in bank
```

**If cash buffer < 3 months**:
- Stop hiring immediately
- Focus on collections (get paid faster)
- Negotiate upfront payments
- Consider raising prices
- Cut unnecessary expenses

## Decision-Making Frameworks

### The $10k Decision Framework

**For decisions that cost <$10k**:
- Decide in <30 minutes
- If ROI > 200%, do it
- If reversible, bias toward yes
- Move fast, iterate

**For decisions that cost >$10k**:
- Sleep on it (24 hours minimum)
- Calculate ROI thoroughly
- Consider opportunity cost
- Get second opinion
- Move deliberately

---

### Opportunity Cost

**Every "yes" is a "no" to something else.**

Taking $5k/month client:
- Uses 20 hours/month
- Those 20 hours can't be used for:
  - $10k/month client (opportunity cost: $5k)
  - Sales/marketing (opportunity cost: future revenue)
  - Systems building (opportunity cost: scalability)

**Ask**: "Is this the best use of these resources?"

---

### The Regret Minimization Framework

**When unsure, ask**:
"Looking back a year from now, will I regret saying yes or saying no more?"

**For client decisions**:
- Regret taking bad client > regret passing on okay client
- Can always get new clients, hard to fire bad ones

**For investment decisions**:
- Regret not investing in growth > regret investing and learning
- Can always make more money, can't get back wasted time
